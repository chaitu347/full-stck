# Full Stack Monorepo

This is a production-level monorepo with:
- **Frontend**: React + Vite (in `apps/web`)
- **Backend**: Node.js + Express (in `apps/api`)
- **Database**: PostgreSQL (Prisma + Neon)
- **Monorepo Tooling**: Turborepo

## Project Structure
- `apps/api`: Express server
- `apps/web`: React frontend
- `packages/database`: Prisma schema and client

## Getting Started
1. Run `npm install`
2. Set up your `.env` file
3. Run `npm run dev`
